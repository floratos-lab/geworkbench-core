Welcome to geWorkbench 2.3.0, an integrative platform for the analysis of genomic and proteomic data.

Major Change: The "Command Area" has been removed from the geWorkbench graphical interface. Analysis, filtering and normalization components can now be invoked from a right-click pop-up menu on any data node in the Project Folders component, or from the "Commands" menubar item. Please see the "Analysis Framework" topic in the built-in Help system, or www.geworkbench.org for more details.
Please see the ReleaseNotes.txt file (in the geWorkbench installation directory) for details on all major changes included in this release.

geWorkbench is delivered with only basic components loaded. To select additional components specific to your own interests, please click on menu bar item Tools->Component Configuration.

Please see http://www.geworkbench.org for a growing list of geWorkbench component tutorials, plus updated documentation, FAQs, and known issues.
