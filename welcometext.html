Welcome to geWorkbench [development], an integrative platform for the analysis of genomic and proteomic data.

Please see the ReleaseNotes.txt file (in the geWorkbench installation directory) for details on all major changes included in this release.

geWorkbench 2.1 is delivered with only basic components loaded. To select additional components specific to your own interests, please click on menu bar item Tools->Component Configuration.

Please see http://www.geworkbench.org for a growing list of geWorkbench component tutorials, plus updated documentation, FAQs, and known issues.
