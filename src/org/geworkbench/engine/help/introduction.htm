The NCICB gene expression analysis workbench

Welcome to the NCICB gene expression analysis workbench!

This software is built on open source technology. Its objective is to provide a comprehensive and extendible collection of analytical, visualization and annotation tools to facilitate working with data from gene expression arrays. Some of the key features include:

  • Support of a variety of input formats. A simple plug-in mechanism allows users to define and use any input format they wish. The present release provides support for:
    • Affymetrix .txt files (MAS 4.0/5.0)
    • Genepix .gpr files.
  • Seamless integration with gene expression data residing at the NCI servers.
  • A variety of filters, normalizers and clustering algorithms. Similar to input formats, a plug-in framework supports the addition of any number of user defined filters/normalizers/algorithms.
  • A rich collection of widgets for visualizing both source data and analysis results.
  • Seamless access to annotation sources and pathways (via NCI's open source caBIO API).

We hope that you will find the tools provided here useful.
