Experiment Information Panel

This non-editable component allows for viewing Experiment Information associated with a Microarray dataset. What constitutes "experiment information" varies, depending on the input source where the microarray set came from. E.g., for Affy input files it is the file preamble (the part just before the data lines which provides various experiment execution parameters) that becomes the experiment information. For datasets obtained from NCI servers, the experiment information is the string returned by the "ExperimentImpl" object representing a dataset.
