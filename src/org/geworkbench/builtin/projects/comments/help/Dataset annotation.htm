Dataset Annotation

The Dataset Annotation pane allows users to attach comments to Project Folders nodes.  The pane is located in the Command and Analysis Area (see picture below). A user first selects the node of interest in the Project Folders tree and then enters the desired annotation (in the form of free text) in the Dataset Annotation pane. Annotations are persisted along with their corresponding Project Folder nodes when the workspace is saved.

Annotate a Dataset

1           Navigate to the Dataset Annotation pane

2           Type in the comments about the dataset in the text area. 

Note:  The entries are not time stamped.
