Merge Dataset

For analysis purposes, it’s desirable in some cases to combine (or "merge") data from different microarray experiments into one set.  Files of the same type can be merged in the project folder (after loaded) or in the File Open window (during the loading process).

To merge data from several loaded experiments:

1.          Select the corresponding nodes from the project tree. By clicking the first file, holding down the shift key, and clicking the last file in the list.

。          Note: The application will highlight the files selected.

2.          Choose File -> Merge Datasets from the main menu. If all the arrays involved in the merge operation are of the same type (same chip technology and same markers) the result will be an aggregate microarray set that comprises all individual arrays.

3.          The project tree will be updated to reflect the merged file with a default name.

To merge files while loading experiments:

Alternatively, any number of data files of the same type can be merged together at the time there are read in using the File Open dialogue.

1.      Right-click on the Project Folder and select the Open File dialogue.

2.      Highlight the files to be read in and check the Merge Files box.

3.      Click Open.

4.      The merged dataset will appear as a single entry in the Project.
